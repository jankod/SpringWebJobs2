About SpringWebJobs

A modern job management system built with Spring Boot

Features

  • Feature
    Job Scheduling
    Schedule jobs to run at specific times or intervals
  • Feature
    Job Monitoring
    Monitor job status and execution details
  • Feature
    Error Handling
    Automatic retry and error notification
  • Feature
    Reporting
    Generate reports on job execution statistics

Technology Stack

Backend
  • Spring Boot
  • Spring Data JPA
  • PostgreSQL
  • Lombok
Frontend
  • Thymeleaf
  • Tabler CSS
  • Bootstrap
  • JavaScript
DevOps
  • Docker
  • Maven
  • Git
